# STILL-inspired Portfolio (GitHub Pages)

Minimalist, image-first portfolio inspired by stillblog.net.

## Deploy
1. Create repo named **<your-username>.github.io**
2. Upload these files to the repo root
3. GitHub → Settings → Pages → Deploy from a branch → main → /(root)
4. Visit https://<your-username>.github.io

## Customize
- Replace the placeholder GitHub/LinkedIn links in `assets/data.js` (and the buttons in `index.html`)
- Replace `#` demo/repo links in `assets/data.js` / `work.html`
